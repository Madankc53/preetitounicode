सामान्य जिज्ञासाहरू

Preeti font भनेको के हो?

Preeti एउटा पुरानो ASCII-based Nepali font हो। Unicode standard आउनुअघि Nepal मा यही font सबैभन्दा बढी प्रयोग हुन्थ्यो। अहिले पनि धेरै सरकारी कार्यालय र newspaper मा प्रयोगमा छ।

मेरो document कहीं save हुन्छ?

हुँदैन। सबै processing तपाईंको आफ्नै browser मा हुन्छ। कुनै पनि text वा file हाम्रो server मा जाँदैन। पूर्ण रूपमा private।

Kantipur र PCS Nepali पनि support गर्छ?

हो। यो tool ले Preeti, Kantipur र PCS Nepali — तीनवटै legacy fonts support गर्छ। dropdown बाट सही font छान्नुस्।

एकैपटक धेरै files convert गर्न मिल्छ?

मिल्छ। File Upload section मा multiple .docx वा .txt files drag & drop गर्नुस् — एकैपटक सबै convert हुन्छ।

Internet नभए पनि काम गर्छ?

गर्छ। browser मा एकपटक site खोलेपछि PWA install गर्नुस् — त्यसपछि offline पनि सबै features काम गर्छन्।
