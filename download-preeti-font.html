⬇️ Install Guide

How to Install Preeti Font on Windows

1

Download the font file

Download Preeti.ttf from a trusted source such as the Font Squirrel or the Government of Nepal font repository.

2

Right-click the .ttf file

Right-click on the downloaded Preeti.ttf file in Windows Explorer and select "Install" or "Install for all users".

3

Restart your application

Close and reopen your Word, Excel, or other application. Preeti will now appear in the font list.

4

Select Preeti font

Open your Preeti document and select Preeti from the font dropdown. The garbled text should now display as correct Nepali.

How to Install Preeti Font on Mac

1

Download Preeti.ttf

Download the Preeti font file to your Mac.

2

Open Font Book

Double-click the Preeti.ttf file. Font Book will open and show a preview of the font.

3

Click "Install Font"

Click the Install Font button. Preeti is now installed and available in all applications.

Why You May Need Preeti Font

You need the Preeti font if you have old documents, reports, or files that were created using the Preeti encoding system. Without the font installed, these documents display as meaningless Latin characters instead of Nepali text.

Common scenarios where Preeti font is needed:

  • Opening old government documents from before 2010
  • Viewing legacy newspaper or publication archives
  • Working with old Word documents or Excel files
  • Printing documents on systems with Preeti-only printers

Preeti Font Variants

Several fonts use the same or similar encoding as Preeti:

  • Preeti — The most common, used in most offices
  • Kantipur — Popular in newspapers and publications
  • Sagarmatha — Used in some government departments
  • Himalaya — Another common legacy font
