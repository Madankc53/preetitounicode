About Preeti Font

What is Preeti font, and why do you need to convert it?

Preeti is a legacy ASCII-encoded Nepali font that dominated Nepali publishing, government offices, and newspapers for over two decades. Today, it is the single biggest barrier to sharing Nepali text across the modern web.

Before Unicode became the international standard for encoding text, Nepali typists used fonts like Preeti, Kantipur, and PCS Nepali. These fonts work by remapping the standard English keyboard: pressing s types क, pressing v types ख, and so on. The text is stored as ordinary ASCII — the Nepali appearance only comes from the font rendering.

The problem: without Preeti font installed, the text appears as random English letters — completely unreadable. Emails, websites, WhatsApp messages, and government portals cannot display Preeti text correctly. Unicode Devanagari, on the other hand, works on every device, every operating system, and every modern application — no font installation required.

Millions of pages of Nepali content — in government documents, newspapers, books, and archives — are still locked in Preeti format. Converting Preeti to Unicode is the essential step for preserving, sharing, and modernizing this content.

FAQ

Frequently asked questions

What is Preeti font and why does it look like English letters?
Preeti is a legacy encoding system that reuses the standard ASCII character set to represent Nepali (Devanagari) characters. The visual appearance depends entirely on the Preeti font being installed. Without it, the stored characters — which are actually English letters — display as-is. Unicode Devanagari is self-contained: the character itself is Nepali, regardless of what font is used.
Is this Preeti to Unicode converter completely free?

Your text and files are never sent to any server. The entire conversion engine runs locally in your browser using JavaScript. This includes .docx file processing — the file is read by your browser, converted, and the result stays on your device. You can verify this by turning off your internet connection and using the tool — it works perfectly offline.
What is the difference between Preeti, Kantipur, and PCS Nepali?
All three are legacy ASCII-based Nepali fonts, but they use different key mappings. In Preeti, 's' = क. In Kantipur, 'k' = क. PCS Nepali is similar to Preeti with minor differences. This tool supports all three — select the correct one from the font dropdown before converting. If your output looks wrong, try switching fonts.
Why is some of my converted text incorrect?
Complex conjunct consonants (जोडाक्षर) and some special characters can be tricky. Use the "Clean Text" button to fix the most common issues like | → ।, extra spaces, and vowel sign ordering. If specific characters are wrong, use the "Report Error" link to let us know — we update the mapping table regularly.
Can I convert Unicode back to Preeti?
Yes. Click "Unicode → Preeti" in the mode toggle. This is useful if you need to edit old Preeti-format files. Note that reverse conversion accuracy is somewhat lower than forward conversion, as some Unicode sequences can map to multiple Preeti key combinations.
Does this work on mobile phones?
Yes, fully. The tool is mobile-optimized and works on Android and iOS. You can also install it as a PWA (Progressive Web App) — tap "Add to Home Screen" in your browser — to use it offline without internet, which is helpful in areas of Nepal with limited connectivity.
Which Unicode font should I use after converting?
Any Unicode Devanagari font works. Popular choices in Nepal include Kalimati (most common, comes with Nepali OS), Mangal (standard in Word on Windows), Noto Sans Devanagari (free from Google, great for web), and Preeti Unicode (looks similar to Preeti). For government documents, Kalimati is the official standard.
