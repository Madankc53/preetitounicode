About This Tool

When do you need Unicode to Preeti conversion?

While Nepal has largely transitioned to Unicode, many legacy systems — older government portals, certain printing workflows, and archival publishing software — still require text in Preeti font format. This tool converts modern Unicode Devanagari back to the ASCII-encoded Preeti format.

Common use cases include: editing old Preeti-format Word documents that must stay in Preeti, submitting text to legacy software or databases that only accept Preeti-encoded input, and generating Preeti output for printing workflows that rely on older Nepali fonts.

Important: After copying the Preeti output into Word or any text editor, you must apply the Preeti font to see Nepali text. Without the Preeti font, it will appear as random English letters — that is normal and expected behavior for this legacy encoding system.

FAQ

Frequently asked questions

Why does the converted text look like English letters?
This is correct behavior. Preeti is a legacy ASCII-encoding system — the text is stored as English characters, and only appears as Nepali when the Preeti font is applied. Copy the output into Word or any text editor and change the font to "Preeti" to see proper Nepali Devanagari text.
Is this converter free?
Yes, completely free. No account, no login, no character limit. All conversion runs in your browser — no data is sent to any server.
Where can I get the Preeti font?
Preeti font is widely available for free download from Nepali government and typography websites. Search "Preeti font download" and install it on your Windows or Mac system. On Windows, right-click the downloaded .ttf file and select "Install".
Why is some text not converting correctly?
Unicode to Preeti reverse conversion is inherently more complex than Preeti to Unicode. Some Unicode sequences — especially rare conjuncts (jodakshar) and certain matras — may not have a perfect Preeti equivalent. The tool handles all common characters accurately. If you find a specific issue, please use our feedback link.
What is the difference between Preeti and Kantipur?
Both are legacy ASCII-based Nepali fonts with different key mappings. Preeti was widely used in general publishing and government. Kantipur was the preferred font in newspapers like Kantipur Daily. Choose the one that matches the software or system you are targeting.
